AIgeam 是一个现代化的 Web 游戏集合平台，包含多种经典和创新游戏。采用最新的 Web 技术构建，提供流畅的游戏体验和美观的用户界面。

## ✨ 特性

- 🎲 多种经典游戏（2048、贪吃蛇、俄罗斯方块等）
- 🏆 排行榜和成就系统
- 📊 游戏数据统计和分析
- 🔒 用户认证和授权
- 📱 响应式设计，适配各种设备
- 🚀 高性能和可扩展性

## 🚀 快速开始

1. 克隆仓库
```bash
git clone https://github.com/yourusername/aigeam.git
cd aigeam
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```

4. 访问 [http://localhost:3000](http://localhost:3000)
